eQTL2GWAS: A Workflow and Data Ressource for Colocalized eQTLs and GWAS
==========================================================================

.. toctree::
   :maxdepth: 2
   :caption: Contents:

   content/workflow

This site gives access to

- A pipeline to colocalize eQTLs and GWAS from the OpenGWAS and EBI eQTL databases
- Colocalization data between ??? eQTLs and ??? GWAS.

The source databases are:

- `eQTL Catalogue <https://www.ebi.ac.uk/eqtl/>`_
- `IEU OpenGWAS project <https://gwas.mrcieu.ac.uk/>`_

The colocalization data can be accessed in these ways: `downloaded <content/download.html>`_ or accessed remotely using:

- `UCSC (Remote) <content/ucsc.html>`_
- `Tabix (Remote and local) <content/scores.html>`_
- `Download of raw TSV files (Local) <content/download.html>`_

The workflow can be run following these instructions:

- `Run the workflow <content/workflow.html>`_
